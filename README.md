# Colombia Fixtures

Aplicación PHP para consultar los próximos partidos de la selección Colombia.

## Requisitos
- PHP 8.0+
- Extensión cURL

## Instalación
1. Configurar `URL_API` en el archivo `.env`.
2. Iniciar el servidor: `php -S localhost:8001 index.php`.

## Estructura
- `index.php`: Punto de entrada y plantilla HTML.
- `logic.php`: Procesamiento de datos y scraping.
- `styles.css`: Estilos de la interfaz.
